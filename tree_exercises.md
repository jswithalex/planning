#### HTML Tree Exercises

Turn a description of a tree structure into HTML

description:
a <span> tag nested inside of a <p> tag

final html:
<p>Today I went to <span>the store<span>.</p>

description:

a <body> tag that contains a <header>, which itself contains an <h2> tag.

html:
<body>
  <header>
    <h2>Web Site</h2>
  </header>
</body>
